# Практическое задания №1: Попова Е.О.

В рамках индивидуального задания была разработана гибридная модель классификации гистологических патчей на 9 классов. Реализация сочетает преимущества сверточных нейронных сетей и классических методов машинного обучения.

## 1. Извлечение признаков с помощью DenseNet121

Взята предобученная нейронная сеть DenseNet121.

Её основные слои заморожены, поэтому модель не обучается заново.

Классификационная часть заменена на Identity(), и сеть работает только как извлекатель признаков (превращает изображение в вектор).

## 2. Классификатор SVM

После того как DenseNet извлекает признаки, они подаются в линейный SVM.

SVM обучается с помощью GridSearchCV, что автоматически подбирает лучший параметр C.

Таким образом получается гибрид: CNN + SVM.

Реализованы функции save() и load(), которые сохраняют/загружают модель автоматически.

Если файла нет — модель скачивается из Google Drive.
